# Employee Onboarding Workflow - Quick Start Guide

## For HR Managers

### Starting a New Employee Onboarding

1. **Navigate to Employee Onboarding**
   - Click "Employee Onboarding" in the sidebar
   - Click "+ New Employee" button

2. **Enter Basic Information**
   - First Name, Last Name, Email (required)
   - Phone, Position, Department
   - Start Date
   - **Home State** (important for licensing)

3. **Click "Create & Start Workflow"**
   - System automatically creates workflow with all tasks
   - Employee appears in "In Progress" list

### Managing Workflow Tasks

1. **View Employee Workflow**
   - Click "View Workflow" on any employee card
   - See all 7 stages and their tasks

2. **Complete Tasks**
   - Navigate to department tabs (HR, Recruiter, Accounting, etc.)
   - Check off completed tasks
   - Add notes if needed

3. **Track Progress**
   - Progress bar shows overall completion %
   - Current stage highlighted
   - Completed stages show checkmark

## For Accounting Managers

### Payroll Setup Tasks

1. **Validate Payroll Information**
   - Go to Accounting tab in workflow
   - Complete all validation tasks:
     - Pay rate validation
     - Pay schedule validation
     - Deductions validation
     - State compliance validation

2. **New State Licensing**
   - If employee is in new state, system triggers licensing workflow
   - Coordinate with Licensing team
   - Track completion in Business Licensing module

3. **Employee Classification**
   - Go to "Employee Classification" section
   - Select type: Billable / Non-Billable / Operational

   **For Billable Employees:**
   - Enter Client Name
   - Enter PO Number
   - Click "Save Classification"

   **For Non-Billable:**
   - Enter Internal Project name

   **For Operational:**
   - Enter Department Assignment

4. **Grant Final Approval**
   - Once all tasks complete, grant Accounting approval
   - This unblocks final sign-off

---

## For Immigration Managers

### Immigration Tracking

1. **Receive Notification**
   - When employee requires sponsorship, you'll see task in Immigration tab

2. **Enter Immigration Details**
   - Go to Immigration Management module
   - Create new immigration record for employee
   - Enter visa type, EAD dates, passport info

3. **Monitor Expiring Documents**
   - Dashboard shows alerts for documents expiring in 30 days
   - System tracks EAD expiry dates
   - Plan renewals accordingly

4. **Grant Approval**
   - Return to employee workflow
   - Grant Immigration approval when ready

---

## For Licensing Managers

### State Licensing Setup

1. **Check Business Licensing Module**
   - Navigate to "Business Licensing" in sidebar
   - See all states requiring setup

2. **Complete Requirements**
   - For each new state, create:
     - State Withholding Account
     - Unemployment Insurance Account
     - Workers' Compensation policy

3. **Update Workflow**
   - Go back to employee workflow
   - Check off completed licensing tasks

4. **Grant Approval**
   - Once all licensing complete, grant Licensing approval

---

## For Recruiters

### Initiating Onboarding

1. **Confirm Project Rate**
   - Verify rate with client
   - Mark "Confirm project rate" task as complete

2. **Coordinate with HR**
   - Ensure HR has sent client confirmation email
   - Track confirmation receipt

3. **Grant Final Approval**
   - Review all stages complete
   - Grant Recruiter approval

## Common Workflows

### Billable Employee (Client Project)
1. Initiate onboarding with home state
2. Complete Stages 1-4
3. If new state ‚Üí Complete licensing (Stage 5)
4. Classify as "Billable"
5. Link to Client and enter PO number
6. Get all 5 department approvals
7. ‚úÖ Employee can now submit timesheets

### Non-Billable Employee (Internal)
1. Initiate onboarding
2. Complete Stages 1-4
3. Skip or fast-track licensing if needed
4. Classify as "Non-Billable"
5. Assign to Internal Project
6. Get all department approvals
7. ‚úÖ Employee can track hours on internal work

### Operational Employee (Department)
1. Initiate onboarding
2. Complete Stages 1-4
3. Skip or fast-track licensing
4. Classify as "Operational"
5. Assign to Department
6. Get all department approvals
7. ‚úÖ Employee can access systems

## Troubleshooting

### "Why can't employee access timesheets?"
Check in this order:
1. Onboarding status must be "Completed"
2. All 5 departments must have approved
3. Classification must be verified
4. Links must be complete:
   - Billable ‚Üí Client + PO
   - Non-Billable ‚Üí Internal Project
   - Operational ‚Üí Department
