## üöÄ Common Tasks

### 1. Change Pricing

**To update monthly price:**
1. Click on plan tab (Starter/Professional/Enterprise)
2. Scroll to "Pricing" card
3. Update "Monthly Price ($)" field
4. Click "Save Changes" at top

**To update annual price:**
1. Same as above, but update "Annual Price ($)" field
2. System automatically shows monthly equivalent and savings %

**Result:** New signups immediately see updated pricing

---

### 2. Enable/Disable Features

**To turn on a feature:**
1. Select plan tab
2. Scroll to "Features & Capabilities" card
3. Find the feature (e.g., "Immigration Management")
4. Toggle switch to ON (blue)
5. Click "Save Changes"

**To turn off a feature:**
1. Same as above
2. Toggle switch to OFF (gray)
3. Click "Save Changes"

**Result:** Feature immediately available/unavailable for new signups on that plan

---

### 3. Mark Plan as Popular

**Steps:**
1. Select plan tab (usually Professional)
2. Find "Mark as Popular" toggle in "Basic Information"
3. Turn ON
4. Click "Save Changes"

**Result:** 
- Plan shows "Most Popular" badge
- Plan card highlighted on signup page
- Recommended for your best-value plan

---

### 4. Hide/Show a Plan

**To hide a plan from customers:**
1. Select plan tab
2. Find "Plan Enabled" toggle in "Basic Information"
3. Turn OFF
4. Click "Save Changes"

**To show a hidden plan:**
1. Same steps, but turn toggle ON

**Result:** Plan immediately hidden/shown on signup page

---

### 5. Update Plan Description

**Steps:**
1. Select plan tab
2. In "Basic Information" card, find "Description" field
3. Update text
4. Click "Save Changes"

**Best practices:**
- Keep under 100 characters
- Highlight who it's for (e.g., "For small teams")
- Mention key benefit

---

### 6. Change Usage Limits

**To update employee limit:**
1. Select plan tab
2. In "Usage Limits" card, find "Maximum Employees"
3. Update number
4. Use 9999 for "unlimited"
5. Click "Save Changes"

**Same process for:**
- Maximum Clients
- Document Storage (text: "10GB", "100GB", "Unlimited")
- Support SLA (text describing support level)

---

### 7. Preview Changes

**Before saving:**
1. Make your changes
2. Scroll to bottom of page
3. View "Preview" card
4. Check how plan will look to customers

**After saving:**
1. Log out (or open incognito window)
2. Click "Sign Up"
3. Navigate to "Choose Plan" step
4. Verify changes appear correctly

## ‚ö†Ô∏è Important Notes

### Things to Know:
- ‚úÖ Changes are immediate (no delay)
- ‚úÖ Only affects NEW signups
- ‚ùå Does NOT change existing customer subscriptions
- ‚ùå Cannot be undone (except "Reset to Defaults")
- üíæ Always click "Save Changes" or changes are lost

### Best Practices:
1. **Test first:** Make small changes, test signup
2. **Document:** Keep notes on pricing decisions
3. **Communicate:** Tell your team about changes
4. **Monitor:** Watch signup conversions after changes

### Warning Signs:
- üö® "Unsaved Changes" badge = you haven't saved yet

## üîß Troubleshooting

### "Changes not showing on signup page"
1. Did you click "Save Changes"?
2. Did you see success message?
3. Try hard refresh (Ctrl+Shift+R or Cmd+Shift+R)
4. Check if plan is enabled

### "Reset to Defaults" button
- ‚ö†Ô∏è **WARNING:** This erases ALL your customizations
- Use only if you really messed up
- Confirms before resetting
- Cannot undo

### Need help?
- Check the Preview card to see what customers see
- Test in incognito window to see actual signup page
- Contact technical support if stuck

```

Steps:
1. Go to Professional tab
2. Scroll to "API Access" feature
3. Toggle ON
4. Save
```

```
Goal: Stop offering Starter plan

Steps:
1. Go to Starter tab
2. Toggle "Plan Enabled" OFF
3. Save

Result: Only Professional and Enterprise shown
Note: Existing Starter customers unaffected
```
