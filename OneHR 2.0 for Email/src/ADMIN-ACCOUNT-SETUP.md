## ğŸ‘¥ Adding Team Members

Once your admin account is created, you can add team members:

### **How to Add Users**

1. **Login** with your admin account
2. **Navigate** to Settings or User Management
3. **Click** "Add User" or "Invite Team Member"
4. **Fill in** user details:
   - Name
   - Email
   - Role (HR Manager, Recruiter, Employee, etc.)
5. **Send** invitation or create account
6. **User receives** login credentials

### **Available Roles**

When creating users, you can assign these roles:

| Role | Access Level | Use Case |
|------|-------------|----------|
| **Admin** | Full access to everything | Organization leaders, founders |
| **HR Manager** | Employee & compliance management | HR department heads |
| **Recruiter** | Onboarding & candidate management | Recruitment team |
| **Accounting Manager** | Financial & invoicing | Finance team |
| **Immigration Team** | Immigration cases & compliance | Immigration specialists |
| **Licensing Team** | Business licenses & renewals | Compliance officers |
| **Employee** | Limited self-service access | Regular employees |
| **Client Admin** | Client portal access | External client contacts |

### **Custom Roles**

Admins can also create custom roles with specific permission sets tailored to their organization's needs.

## ğŸ”’ Security & Access Control

### **Role-Based Access Control (RBAC)**

The platform uses RBAC to ensure users only see what they need:

- **Admin**: Sees and controls everything
- **HR Manager**: Employee data, onboarding, documents
- **Recruiter**: Candidate pipeline, new hires
- **Accounting**: Timesheets, invoices, expenses
- **Employee**: Personal profile, timesheets, documents

### **Password Management**

- **Default Passwords**: New users can be assigned temporary passwords
- **First Login**: Users prompted to change password
- **Reset**: Admins can reset user passwords
- **Security**: Minimum 8 characters required

## ğŸ†˜ Troubleshooting

### **Issue: Can't Add Users**

**Solution**: Verify you're logged in as admin. Only admin users can create new accounts.

### **Issue: User Can't Login**

**Solution**: 
1. Check user status (must be "Active")
2. Verify email address is correct
3. Reset password if needed
4. Check role assignment

### **Issue: User Has Wrong Permissions**

**Solution**:
1. Go to User Management
2. Find the user
3. Edit their role
4. Save changes
5. User may need to logout/login
