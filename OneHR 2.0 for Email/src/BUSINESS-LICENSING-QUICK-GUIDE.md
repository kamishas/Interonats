## âš¡ Quick Actions

### **Check for Expiring Licenses**
1. Click **"Check Reminders"** button in the header
2. System scans all licenses
3. Creates tasks for licenses at 120/90/60/30 day thresholds
4. View results in toast notification

### **Create a Renewal Task**
1. Go to **All Licenses** tab or **Overview** tab
2. Find the license you want to renew
3. Click **"Create Task"** button
4. Fill out the form:
   - **Assign to Department**: Select Licensing, Accounting, Legal, etc.
   - **Assign to Person**: (Optional) Specific person's name/email
   - **Due Date**: When renewal must be completed
   - **Priority**: Low, Medium, High, or Urgent
   - **Upload Proof** (optional):
     - Proof of Renewal (renewed license PDF)
     - Proof of Payment (receipt PDF)
5. Click **"Create Task"**

### **Complete a Renewal Task**
1. Go to **Renewal Tasks** tab
2. Find your task in the list
3. Click **"Complete"** button
   - OR Click **"Details"** â†’ **"Mark Complete"**
4. Task moves to completed status
5. Audit trail automatically logs the completion

### **View License History**
1. Go to **All Licenses** tab
2. Click on any license card
3. Dialog opens with 2 tabs:
   - **Audit Trail**: Complete history of all actions
   - **Related Filings**: Annual reports, UI returns, insurance renewals

### **Add a Related Filing**
1. Click on a license
2. Switch to **Related Filings** tab
3. Click **"Add Related Filing"**
4. Fill out the form:
   - **Filing Type**: Annual Report, UI Return, Insurance Renewal, Tax Filing, or Other
   - **Filing Name**: e.g., "2024 Annual Report"
   - **Due Date**: When filing is due
   - **Fee**: (Optional) Filing cost
   - **Notes**: (Optional) Additional details
5. Click **"Add Filing"**

## ğŸ†˜ Troubleshooting

### **Reminder Not Created**
- Verify reminder system is enabled (Settings)
- Check if license expiry date is set
- Ensure license status is not "archived"
- Run manual "Check Reminders"

### **Can't Upload Document**
- Verify file is PDF format
- Check file size (must be < 10MB)
- Refresh page and try again
- Contact admin if persistent

### **Task Not Showing**
- Check if filtering is applied
- Verify task wasn't cancelled
- Refresh the page
- Check correct license ID

### **Audit Trail Empty**
- New licenses won't have history yet
- Only actions after implementation are logged
- Check if looking at correct license
